## 8. 폴더 구조

### 📂 전체 프로젝트 구조

```
HanaPath/
├── 📁 BE/                    # 백엔드
├── 📁 FE/                    # 프론트엔드
├── 📁 realtime/              # 실시간 시세
├── 📁 infrastructure/        # 인프라
├── docker-compose.yml
└── README.md
```

<details>
<summary><b> 📁 BE/backend - Spring Boot 백엔드</b></summary>
<br>

```
BE/backend/
├── 📁 src/main/java/com/hanapath/backend/
│   ├── 📁 account/            # 투자 계좌 관리
│   ├── 📁 attendance/         # 출석 체크
│   ├── 📁 chatbot/            # 챗봇
│   ├── 📁 community/          # 커뮤니티
│   ├── 📁 config/             # 전역 설정
│   ├── 📁 email/              # 이메일 인증
│   ├── 📁 hanamoney/          # 하나머니 
│   ├── 📁 investment/         # 모의 투자
│   ├── 📁 news/               # 금융 뉴스
│   ├── 📁 notification/       # 알림 시스템
│   ├── 📁 quiz/               # 금융 퀴즈
│   ├── 📁 savings/            # 목표 저축
│   ├── 📁 smishing/           # 스미싱 탐지
│   ├── 📁 stock/              # 주식 시세
│   ├── 📁 store/              # 상품 스토어
│   ├── 📁 users/              # 사용자 인증
│   └── 📁 wallet/             # 전자 지갑
│
├── 📁 src/main/resources/     # 환경 설정
├── 📁 crawler/                # 뉴스 크롤러
├── build.gradle
├── Dockerfile
└── docker-compose.yml
```

> **Spring Boot 3.x, JPA, Spring Security, JWT, MySQL, MongoDB, Redis**

</details>

<details>
<summary><b> 📁 BE/model - AI 스미싱 탐지 모델</b></summary>
<br>

```
BE/model/
├── smishingmodel.h5               # 스미싱 모델
├── tokenizer.pkl                  # 토크나이저
├── smishing_detection_service.py  # Flask 서버
├── requirements.txt
└── Dockerfile
```

> **TensorFlow, Keras, Flask - 문자 메시지 스미싱 판별**

</details>

<details>
<summary><b> 📁 FE - Next.js 프론트엔드</b></summary>
<br>

```
FE/
├── 📁 app/                     # 페이지 라우팅
│   ├── 📁 api/                 # API 라우트
│   ├── 📁 community/           # 커뮤니티
│   ├── 📁 hanamoney/           # 하나머니
│   ├── 📁 investment/          # 모의 투자
│   ├── 📁 news/                # 금융 뉴스
│   ├── 📁 portfolio/           # 포트폴리오
│   ├── 📁 store/               # 상품 스토어
│   ├── 📁 wallet/              # 전자 지갑
│   ├── 📁 mypage/              # 마이페이지
│   ├── 📁 login/               # 로그인
│   └── 📁 signup/              # 회원가입
│
├── 📁 components/              # UI 컴포넌트
│   ├── 📁 chatbot/             # 챗봇
│   ├── 📁 charts/              # 차트
│   ├── 📁 community/           # 커뮤니티
│   ├── 📁 dashboard/           # 대시보드
│   ├── 📁 mypage/              # 마이페이지
│   ├── 📁 security-search/     # 보안 검색
│   ├── 📁 wallet/              # 전자 지갑
│   └── 📁 ui/                  # 공통 컴포넌트
│
├── 📁 lib/                     # 유틸리티
│   └── 📁 api/                 # API 클라이언트
│
├── 📁 hooks/                   # Custom Hooks
├── 📁 store/                   # 상태 관리
├── 📁 types/                   # 타입 정의
├── 📁 public/                  # 정적 파일
├── package.json
├── tsconfig.json
└── Dockerfile
```

> **Next.js 14, TypeScript, Tailwind CSS, shadcn/ui, Zustand**

</details>

<details>
<summary><b> 📁 realtime - 실시간 주식 시세</b></summary>
<br>

```
realtime/
├── kis_ws_client.py       # WebSocket 클라이언트
├── kis_chart_api.py       # 차트 API
├── requirements.txt
└── Dockerfile
```

> **한국투자증권 OpenAPI - 실시간 주식 시세**

</details>

<details>
<summary><b> 📁 infrastructure - 인프라 구성</b></summary>
<br>

```
infrastructure/
├── 📁 nginx/              # 웹 서버 설정
└── 📁 scripts/            # 배포 스크립트
```

> **Nginx, SSL/TLS, 배포 자동화**

</details>
